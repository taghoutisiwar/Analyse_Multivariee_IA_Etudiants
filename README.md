# Analyse Multivariée — Impact de la dépendance à l’IA sur les performances académiques

Projet réalisé dans le cadre du module :
Méthodes statistiques et étude de données.

## Objectif

Ce projet analyse l’impact de l’utilisation de l’intelligence artificielle sur les performances académiques des étudiants à travers des méthodes d’analyse multivariée.

Les techniques utilisées :

- Analyse en Composantes Principales (ACP)
- Clustering K-means
- Analyse descriptive
- Matrice de corrélation
- Heatmap des clusters

## Dataset

Dataset utilisé :
Academic Outcomes & AI Dependency Analysis Dataset

Source Kaggle :
https://www.kaggle.com/datasets/abdullahmeo/academic-outcomes-and-ai-dependency-analysis-dataset

## Technologies utilisées

- R
- FactoMineR
- factoextra
- tidyverse
- corrplot
